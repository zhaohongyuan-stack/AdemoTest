# 开发日志

## 2026-06-11 - 采购管理模块开发

### 开发内容
1. **基础架构搭建**
   - 创建枚举类：OrderStatusEnum（订单状态）、StockTypeEnum（库存类型）
   - 创建异常体系：ErrorCode、BusinessException、GlobalExceptionHandler
   - 创建工具类：OrderNoGenerator（订单号生成器）
   - 创建DTO/VO：PurchaseOrderCreateDTO、PurchaseOrderVO

2. **采购订单核心业务实现**
   - 创建采购单：自动生成单号(PO+日期+序号)、计算总金额、保存主表和明细
   - 采购单入库：状态校验、事务控制、更新库存、生成流水记录
   - 取消采购单：状态校验、更新状态为已取消

3. **API接口**
   - POST /api/purchase-order - 创建采购单
   - PUT /api/purchase-order/{id}/receive - 采购单入库
   - PUT /api/purchase-order/{id}/cancel - 取消采购单
   - GET /api/purchase-order/{id} - 查询详情（含明细）

### 业务规则
- 订单状态：0-待入库、1-已入库、2-已取消
- 只有待入库状态的订单才能入库或取消
- 入库时自动增加库存并生成流水记录(type=1采购入库)
