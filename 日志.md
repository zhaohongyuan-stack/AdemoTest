# 开发日志

## 2026-06-11 - 采购管理模块开发

### 开发内容
1. **基础架构搭建**
   - 创建枚举类：OrderStatusEnum（订单状态）、StockTypeEnum（库存类型）
   - 创建异常体系：ErrorCode、BusinessException、GlobalExceptionHandler
   - 创建工具类：OrderNoGenerator（订单号生成器）
   - 创建DTO/VO：PurchaseOrderCreateDTO、PurchaseOrderVO

2. **采购订单核心业务实现**
   - 创建采购单：自动生成单号(PO+日期+序号)、计算总金额、保存主表和明细
   - 采购单入库：状态校验、事务控制、更新库存、生成流水记录
   - 取消采购单：状态校验、更新状态为已取消

3. **API接口**
   - POST /api/purchase-order - 创建采购单
   - PUT /api/purchase-order/{id}/receive - 采购单入库
   - PUT /api/purchase-order/{id}/cancel - 取消采购单
   - GET /api/purchase-order/{id} - 查询详情（含明细）

### 业务规则
- 订单状态：0-待入库、1-已入库、2-已取消
- 只有待入库状态的订单才能入库或取消
- 入库时自动增加库存并生成流水记录(type=1采购入库)

## 2026-06-11 - 库存管理模块 & 统计查询模块开发

### 开发内容
1. **库存管理模块**
   - 分页查询当前库存：支持按商品名称模糊查询，返回商品ID、编码、名称、当前库存、预警值、是否预警
   - 库存预警查询：查询所有当前库存低于预警值的商品列表
   - 库存流水查询：支持按商品ID、流水类型、时间范围筛选

2. **统计查询模块**
   - 今日采购金额统计：查询今日所有已入库采购单的总金额和订单数
   - 今日销售金额统计：查询今日所有已出库销售单的总金额和订单数
   - 商品销量排行榜：按指定时间范围销售数量倒序排列，返回前10名商品

3. **DTO/VO新增**
   - CurrentStockVO：库存查询VO（含预警标识）
   - StockRecordQueryDTO：库存流水查询条件
   - StatisticsVO：统计结果VO
   - ProductSalesRankVO：商品销量排行VO

### API接口
- GET /api/stock/page - 分页查询当前库存
- GET /api/stock/warning - 库存预警列表
- GET /api/stock/record/page - 分页查询库存流水
- GET /api/statistics/purchase/today - 今日采购统计
- GET /api/statistics/sale/today - 今日销售统计
- GET /api/statistics/product/rank - 商品销量排行榜

### 业务规则
- 库存预警判断：当前库存 < 商品预警值
- 统计仅计算已完成状态（已入库/已出库）的订单
- 销量排行默认返回前10名，支持时间范围筛选
